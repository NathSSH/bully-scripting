Create a ped (a character) with a custom position, as opposed to a pre-defined point like PedCreatePoint.
The ped will be tied to the current script so that it is cleaned up when the script terminates, and will of course be returned for you to script with.

Arguments

  • Ped model ID
  • X coordinate
  • Y coordinate
  • Z coordinate

Results

  • Ped

Example

Create and recruit Algernon as an ally.

local x, y, z = PlayerGetPosXYZ()
local ped = PedCreateXYZ(4, x + 1, y, z)
PedRecruitAlly(gPlayer, ped)
