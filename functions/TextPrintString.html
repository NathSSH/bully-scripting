Prints DEBUG text for a specified amount of time in a specific type.
This is different from TextPrint, as this takes any text as input, instead of IDs. 1 is Objective-styled, 2 is Subtitle-styled.
Note: On all SE versions, both are printed in red colour.

Arguments

  • Text
  • Seconds
  • Text Type

Results

None.

Example

Print "Hello World." for 1 second, in the style of the in-game subtitles.

TextPrintString("Hello World.", 1, 2)
