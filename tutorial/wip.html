Introduction

Hiya, welcome to my Bully script modding tutorial. This tutorial is designed to help you go from knowing nothing about Lua scripting to making your own mods. For each section, an example will be provided as well as some exercises for you to help make sure you understand the section.

In this section, your main task will be to get a custom script running in-game. There are two ways to make script mods, so feel free to skip to the one you are interested in.

  • Scripts.img: your mods will replace the game's normal scripts by modifying the Scripts.img file.
  • derpy's script loader (DSL): your mods will use my script loader to run custom scripts (recommended).

Scripts.img Modding

For a long time, this was the only way to make a script mod. Sometimes it is the better option for a mod, but it is often times just more hassle nowadays.

First let's pick up some tools.

  • Notepad++: the recommended program for making your Lua scripts.
  • luac.exe: a custom Lua 5.0.2 compiler compiled to use similar settings to Bully.
  • img.exe: a basic IMG tool for Bully, though a bit dated it is still often the simplest.

Picking a Script

You could replace any script in the game with your own, but only the 2 most common choices will be covered here.

  • ArcRace1.lur: a popular choice due to it being easily started by playing the arcade machine in the Boy's dorm.
  • STimeCycle.lur: another popular choice due to it being the simplest script that runs non-stop.
