Returns an iterator function that will return each valid ped one at a time.

Arguments

None.

Results

  • Iterator

Example

Make all peds fight the player on command.

SetCommand("fight_me", function()
	for ped in AllPeds() do
		if ped ~= gPlayer then
			PedSetPedToTypeAttitude(ped, 13, 0)
			PedSetEmotionTowardsPed(ped, gPlayer, 0)
			PedAttackPlayer(ped, 3)
		end
	end
end)
